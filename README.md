# openfoam-axial-fan-sim
Transient rotating-mesh CFD simulation of a 90mm, 12-blade self-induced axial fan (4000 RPM) in OpenFOAM-13. Hybrid meshing, Non-Conformal Coupling (NCC), rigid-body mesh motion, and k-ω SST turbulence to resolve unsteady blade-wake interaction and thrust in an open, pressure-driven domain.
